## Engineering Culture

### Tech Stack

**Backend:** Python, Go, Rust (for performance-critical components)
**Frontend:** React, TypeScript
**Infrastructure:** Kubernetes, Terraform, AWS/GCP/Azure
**Data:** PostgreSQL, Redis, Kafka
**ML:** PyTorch, TensorFlow, Ray

### Engineering Principles

1. **Operational excellence:** We own our code in production
2. **Customer obsession:** Every feature ties to measurable customer value
3. **Cost awareness:** GPU hours are expensive - optimize by default
4. **Security first:** Enterprise customers require zero-trust architecture

### Development Velocity

- **Sprint cadence:** 2-week sprints
- **Release cycle:** Weekly releases, minor versions
- **Feature freeze:** 2 weeks before major releases
- **Code review:** Required, 2+ approvals for infrastructure changes
