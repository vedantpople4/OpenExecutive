crawl (OpenExec is a CLI tool). No-capture path per Step 1.

All visuals for this video are code-composed (HTML/CSS/SVG), invented per frame:

  their real sub-agent names (see visible-text.txt for the exact list).
- **Terminal component** — styled monospace terminal window with typewriter
  reveal of the real CLI command and real plain-text output lines.
- **Research-mix dial/gauge** — an SVG gauge animating to the real web:0.7 /
  kb:0.3 ratio.
- **Report-excerpt text cards** — kinetic typography presenting the real
  board_report.md excerpt (dissent, final priority actions with owners/timeframes).
- **Wordmark / title card** — "OpenExec" typography lockup for open/close.

No logo file exists for OpenExec; the title card uses typography only, no
invented logo mark.
